API Reference

This engine exposes a stateless simulation via query parameters. Same inputs always produce identical outputs.

Frame mode:

?mode=frame&skill=0.75&seed=42&isLastFrame=0

Game mode:

?mode=game&skill=0.75&seed=42

Visualization:

?mode=viz&skill=0.75&seed=42
